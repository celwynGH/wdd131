Website Planning Document – W05 Project

Site Name

Sinap K’s Catering Services

1. Naming Conventions

All filenames are lowercase and use hyphens (no spaces).

Example: index.html, style.css, script.js, images/hero.jpg.

2. HTML Validation

The HTML file was checked using the W3C Markup Validation Service. No major errors were found, ensuring semantic and valid HTML5.

3. CSS Validation

The CSS file was validated using the W3C CSS Validator. No major errors, ensuring cross-browser compatibility.

4. Site Name

Sinap K’s Catering Services — This name reflects the family brand, focusing on catering events at affordable prices.

5. Site Purpose

To showcase catering services, menus, packages, and contact information. Customers can browse packages, view past event galleries, and quickly contact us for booking.

6. Scenarios

  • Scenario 1: A customer planning a birthday party wants to know catering packages within their budget.
  • Scenario 2: A couple preparing for a wedding wants to view past catering events and contact us for a customized package.

7. Color Schema

  • #b30000 (Deep Red): Header, footer, navigation
  • #ffcc00 (Gold/Yellow): Buttons and highlights
  • #333 (Dark Gray): Text readability

8. Color Schema Used on Document

This document is styled with #b30000 for headers and accents. Buttons use #ffcc00 for emphasis.

9. Typography

Font Family: Arial, sans-serif

  • Headers (h1, h2): Bold Arial
  • Paragraphs: Regular Arial

10. Typography Used on Document

This planning document uses Arial throughout for consistency.
